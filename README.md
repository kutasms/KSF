# Kuta Service Framwork (KSF)
![](https://img.shields.io/badge/license-Apache%202-blue)

基于AKKA的分布式集群框架，包括对Akka-Http(http1.1)、Java基础类库、缓存（Redis）封装、数据库（Mysql-mybatis和Mongodb）封装等。

使用KSF您将可以快速开发基于Akka的应用程序.
